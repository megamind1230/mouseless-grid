* mouseless-grid

  On-screen grid clicker for Linux. Type 1-9 to zoom into a 3×3 grid, click at the target.

** Features

  - ImageJ-style zoom: 1-9 keys recursively divide screen into 3×3 cells
  - Auto-stops when cell width ≤ minCellPx (default 10px)
  - Left click (LCtrl) / Right click (LAlt) on the final cell
  - Enter confirms early without max zoom
  - Backspace to undo one zoom level
  - Transparent click-through overlay (X11 ARGB visual + xfixes shape input)
  - Full-screen, always-on-top, skip-taskbar
  - Built-in bitmap font (4×6) for cell labels
  - Pure Go, single binary, no cgo
  - Works on X11; transparency requires a compositor (picom, xcompmgr, etc.)
  - XTest WarpPointer for cursor movement + uinput for click
  - Keyboard-only interaction (no mouse clicks on the overlay itself)

** Tech Stack

  | Layer        | Library                          | Purpose                                |
  |--------------+----------------------------------+----------------------------------------|
  | Keyboard     | golang-evdev                     | Read /dev/input/event* devices         |
  | Mouse        | bendahl/uinput + XTest           | Virtual mouse click + cursor warp      |

  | Logging      | logrus                           | Structured file logging                |

** Dependencies (system)

  These must be present on the system for the app to work:

  | Dependency          | Why                              | How to set up                                                     |
  |---------------------+----------------------------------+-------------------------------------------------------------------|
  | X11 (Xorg)          | Overlay window                   | Already running if you're on a Linux desktop                       |
  | evdev kernel module | Read keyboard events             | Already loaded on any modern Linux                                 |
  | uinput kernel module| Virtual mouse device             | =sudo modprobe uinput= (add to /etc/modules-load.d/ to persist)    |
  | SHAPE extension     | Click-through overlay            | Built into Xorg, always available                                  |
  | XFIXES extension    | Input region for click-through   | Built into Xorg, always available                                  |
  | Go 1.26+            | Building from source             | Install: =sudo apt install golang= or =sudo pacman -S go=         |
  | Compositor (optional)| Window transparency              | =sudo apt install picom= / =sudo pacman -S picom=, then add to WM startup |

** Quick Start

  #+begin_src bash
  # Clone
  git clone git@github.com:megamind1230/mouseless-grid.git
  cd mouseless-grid

  # Build
  go build -o mouseless-grid .

  # Run (needs read access to /dev/input and /dev/uinput)
  sudo ./mouseless-grid
  #+end_src

** How to Run

  #+begin_src bash
  # Build from source
  go build -o mouseless-grid .

  # Run with defaults
  sudo ./mouseless-grid

  # Run with custom config
  sudo ./mouseless-grid --config /path/to/config.yaml

  # List available keyboard devices
  sudo ./mouseless-grid --list-devices

  # Verbose debug output
  sudo ./mouseless-grid --debug

  # Generate default config file
  sudo ./mouseless-grid --gen-config

  # Save rendered overlay to PNG for inspection
  sudo ./mouseless-grid --save-debug

  # Diagnostic mode: create overlay, print window ID, sleep 10s
  sudo ./mouseless-grid --check

  # Build + run in one step (no binary left behind)
  sudo go run . --debug
  #+end_src

** Usage

  1. Run the app — full-screen transparent overlay appears with 3×3 grid labelled 1-9
  2. Press 1-9 — the selected cell highlights, region zooms to that cell (3×3 sub-grid shown)
  3. Repeat to refine position until cell is small enough (or press Enter to confirm early)
  4. Press LCtrl for left-click or LAlt for right-click — click fires at cell center, overlay hides, app exits
  5. Press Backspace — undo last zoom step
  6. Press Escape — cancel and exit

** Grid Layout

  Zoom grid (fixed 3×3, number keys):
  #+begin_example
  1  2  3
  4  5  6
  7  8  9
  #+end_example

  Each zoom level subdivides the current region into another 3×3.
  Zoom stops when cell width ≤ minCellPx (configurable, default 10px).
  At that point the state automatically switches to Ready — press LCtrl/LAlt to click.

  Click coordinate = center of the final cell.

** Config

  Default location: ~/.config/mouseless-grid/config.yaml

  #+begin_src yaml
  minCellPx: 10        # stop zooming when cell width ≤ this
  bgColor: "#1a1a2e"   # background color (hex)
  textColor: "#e0e0e0" # label color
  highlightColor: "#00d4ff" # selected cell color
  opacity: 0.4         # window opacity (via _NET_WM_WINDOW_OPACITY)
  logPath: "~/magnus/mouseless-grid/logs/"
  #+end_src

** Permissions

  The app needs:
  - Read access to /dev/input/event* (keyboard devices)
  - Write access to /dev/uinput (virtual device creation)

  Setup without root:
  #+begin_src bash
  sudo groupadd --system uinput
  sudo usermod -a -G input,uinput $USER
  sudo tee /etc/udev/rules.d/99-mouseless-grid.rules <<EOF
  KERNEL=="uinput", GROUP="uinput", MODE="0660"
  EOF
  # Reboot or: sudo modprobe uinput
  #+end_src
