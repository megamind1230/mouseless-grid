* mouseless-grid — Implementation Plan

  On-screen grid clicker for Linux. Type 1-9 to zoom into cells,
  i/o to click at that point.

  Status key: [X] done, [-] in progress, [ ] not started, [~] changed from original plan

** Phase 6: Grid Logic + State Machine

  - [X] =6.1= Zoom model: each press subdivides region into 3×3
    - [X] Auto-advance to Ready when cell width ≤ minCellPx
  - [X] =6.2= Key mapping: num (1-9) or home (asdfghjkl) layouts
  - [X] =6.3= State machine
    - [X] IDLE → (number) → ZOOMED
    - [X] ZOOMED → (number) → ZOOMED (or READY if precision reached)
    - [X] ZOOMED → (backspace) → previous level (or IDLE if empty)
    - [X] ZOOMED → (enter) → READY (early confirm)
    - [X] ANY → (i) → left click + exit
    - [X] ANY → (o) → right click + exit
    - [X] READY → (backspace) → ZOOMED (undo)
    - [X] any → (escape) → exit
  - [X] =6.4= clickX/Y = center of current cell region (updated on every zoom)
  - [X] =6.5= Render grid to image.NRGBA (cells, lines, highlight, labels)
