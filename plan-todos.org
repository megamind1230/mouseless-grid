* mouseless-grid — Implementation Plan

  On-screen grid clicker for Linux. Type characters to select cells, click at that point.
  Architecture follows jbensmann/mouseless (evdev + uinput), with X11 overlay via xgb.

** Phase 6: Grid Logic + State Machine

  - [X] =6.1= Calculate grid dimensions from screen size + targetSubCellPx
    - [X] subGridCols = max(3, screenWidth / (3 * targetSubCellPx))
    - [X] subGridRows = max(3, screenHeight / (3 * targetSubCellPx))
  - [X] =6.2= Map keys to grid cells (dynamic based on grid size)
    - [X] Main grid: j,k,l,u,i,o,m,.,'
    - [X] Sub grid: nearby keyboard keys, generated per size
  - [X] =6.3= State machine
    - [X] IDLE → (key press) → MAIN_SELECTED
    - [X] MAIN_SELECTED → (sub-grid key) → DONE
    - [X] MAIN_SELECTED → (backspace) → IDLE
    - [X] any → (escape) → IDLE
  - [X] =6.4= Calculate pixel coordinates from cell indices
  - [X] =6.5= Render grid to image.NRGBA (cells, lines, highlight)
