* mouseless-grid — Implementation Plan

  On-screen grid clicker for Linux. Type 1-9 to zoom into cells, 
  LCtrl/LAlt to click at that point.
  Architecture: evdev + uinput + X11 overlay via jezek/xgb.

  Status key: [X] done, [-] in progress, [ ] not started, [~] changed from original plan

** Phase 6: Grid Logic + State Machine

  - [X] =6.1= Calculate grid state from selections + screen size + minCellPx
    - [~] Zoom model: each press of 1-9 subdivides region into 3×3
    - [~] Auto-advance to StateReady when cell width ≤ minCellPx
  - [X] =6.2= Map keys to grid cells
    - [~] Fixed mapping: 1-9 → 3×3 grid (not dynamic per screen size)
    - [~] No sub-grid key mapping — pure recursion zoom
  - [X] =6.3= State machine
    - [X] IDLE → (number 1-9) → ZOOMED
    - [X] ZOOMED → (number 1-9) → ZOOMED (or READY if precision reached)
    - [X] ZOOMED → (backspace) → previous level (or IDLE if empty)
    - [X] ZOOMED → (enter) → READY (early confirm)
    - [X] READY → (LCtrl) → left click + exit
    - [X] READY → (LAlt) → right click + exit
    - [X] READY → (backspace) → ZOOMED (undo)
    - [X] any → (escape) → exit
  - [X] =6.4= Calculate pixel coordinates from cell indices
    - [X] clickX/Y = center of final cell region
  - [X] =6.5= Render grid to image.NRGBA (cells, lines, highlight, labels)
