* mouseless-grid — Changelog

  All notable changes to this project are documented here.

** [0.2.0] - 2026-07-18

  Wayland dual-backend, click keys, grid layouts, input refactor.

  Added:
  - [X] Wayland support: wlr-layer-shell overlay + wlr-virtual-pointer mouse
  - [X] Input via wl_keyboard protocol on Wayland (no evdev grab)
  - [X] /i/ left-click, /o/ right-click (replaces LCtrl/LAlt)
  - [X] Clicks work from any zoom level (no Enter-first requirement)
  - [X] Grid layout flag: --grid-layout num (1-9) or home (asdfghjkl)
  - [X] Uppercase glyphs A-Z in bitmap font
  - [X] Per-keypress INFO logging for debugging
  - [X] Device.Close() for proper keyboard cleanup on exit

  Fixed:
  - [X] Wayland "broken pipe" crash — dispatch goroutine + AckConfigure
  - [X] Wayland overlay transparency — opacity passed to Render
  - [X] Wayland surface sizing — all-4 anchors + exclusive zone -1, ignore configure size
  - [X] Enter leak on exit — replaced evdev grab with wl_keyboard on Wayland
  - [X] K and D glyphs in bitmap font
  - [X] color.NRGBA.RGBA() premultiplied value issue in config tests

  Changed:
  - [X] Click keys: i/o only, removed leftctrl/leftalt
  - [X] clickX/clickY set on every zoom step (not just at precision)
  - [X] Keyboard log: presses at INFO, releases at DEBUG

** [0.1.0] - 2026-07-16

  Initial release.

  Added:
  - [X] 3×3 zoom grid with recursive ImageJ-style subdivision
  - [X] Number keys 1-9 for cell selection during zoom
  - [X] LCtrl left-click / LAlt right-click at final cell
  - [X] Enter early-confirm, Backspace undo, Escape cancel
  - [X] Full-screen transparent overlay (X11 ARGB visual)
  - [X] Click-through via xfixes shape extension (empty input region)
  - [X] Window opacity via _NET_WM_WINDOW_OPACITY
  - [X] Always-on-top, skip-taskbar, all-desktops
  - [X] evdev keyboard grab + read loop
  - [X] uinput virtual mouse for click events
  - [X] XTest WarpPointer for cursor positioning
  - [X] 4×6 bitmap font for cell labels (grid/font.go)
  - [X] YAML configuration with defaults
  - [X] CLI flags: --debug, --config, --list-devices, --version, --gen-config, --save-debug, --check
  - [X] Structured logging to file (~/magnus/mouseless-grid/logs/)
  - [X] Expose event handling for re-render after window damage
